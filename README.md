# Mathematical Foundations of Machine Learning

## Abstract
This repository implements machine learning algorithms from mathematical first principles, emphasizing derivations and proofs.

## Contents
1. **Linear Algebra** - PCA, SVD, Matrix Factorizations
2. **Probability** - Bayesian Inference, Gaussian Processes
3. **Optimization** - Gradient Methods, Convex Optimization
